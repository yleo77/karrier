## karrier

Just another simple file uploader.

A dependence package in our team work.

## Usage

### With Command Line

```
karrier -n env_test
```

Note: Like the config of eslint, the config of Karrier is written in `.karrierconfig.js`

### API

```javascript
var karrier = require("karrier");
var config = {};
karrier(config);
```
